# 高中物理甲种本（第一册）重排本

本项目是对高中物理甲种本（第一册）的致敬。虽然该书年代久远（1983—1985年出版），但是内容体系安排比如今的高中教材完整且合理。

本重排本是根据网络上找到的此书扫描版电子文档，使用LaTeX制作而成的重排本电子文档（PDF格式）。书中的矢量图片采用电子版教材中的矢量图，或采用Tikz制作而来。其余的点阵图则是来自于扫描版电子文档（限于作者的能力和精力，无法将所有内容均以矢量图全部重绘）。

注意：本项目的内容勿用于商业目的。
